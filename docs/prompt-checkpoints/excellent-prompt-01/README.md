## 5. 이후 개발 원칙

앞으로의 개선은 반드시 **"우수 프롬프트 1"**에서 분기합니다.

- 4번의 판정력과 압축력을 훼손하지 않음
- 3번의 시간 서사와 인간적 인과만 선택적으로 추가
- 5번처럼 식상=자식, 관성=남편 등 육친·십성 항목식 해설로 회귀하지 않음
- 문장을 육친 공식으로 시작하지 않음
- 십성을 사전식으로 설명하지 않음
- 남편=관성, 자식=식상 같은 일반론을 중심축으로 삼지 않음
- 명리 요소를 실제 삶의 사용 구조로 번역
- 강한 힘과 잘 쓰는 능력을 구분
- 욕구, 통로, 결과를 분리
- 여러 요소를 하나의 핵심 판정으로 압축
- 좋은 프롬프트가 새로 나오면 "우수 프롬프트 2, 3, 4…"로 체크포인트를 추가 보존
