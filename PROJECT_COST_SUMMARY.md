# 휴먼프리즘(Human Prism) 프로젝트 비용 정리 및 운영 계획

## 1. 초기 개발 투자 (2026년 초~중반)

### 1.1 Manus 플랫폼 크레딧 사용
- **총 사용액**: 약 1,500,000원 (150만 원)
- **주요 용도**:
  - 웹앱 개발 및 배포 (React + Express + tRPC + Tailwind)
  - 카카오/네이버 OAuth 통합
  - Claude AI API 연동 (사주 상담 엔진)
  - 벡터 DB 기반 RAG (전문 서적 OCR 처리)
  - 포트원 결제 연동 (준비 단계)
  - 개발 중 LLM 호출 및 테스트

### 1.2 외부 서비스 초기 비용
- **토스페이먼츠**: 초기 개설 비용 (정확한 금액 확인 필요)
- **도메인**: human-prism.com 구매 (연간 비용)
- **SSL 인증서**: 자동 갱신 (Manus 내장)

## 5. 비용 최적화 전략

### 5.1 단기 (1~3개월)
- Claude API 호출 최적화 (프롬프트 길이 단축)
- RAG 청크 크기 조정 (불필요한 조회 감소)
- 캐싱 전략 도입 (반복 상담 응답 재사용)

### 5.2 중기 (3~6개월)
- 로컬 LLM 검토 (비용 절감 vs 품질 트레이드오프)
- 상담 유형별 비용 분석 (어떤 상담이 수익성 높은지)
- 마케팅 비용 계획 (사용자 확보)

### 5.3 장기 (6개월 이상)
- 자체 미니 LLM 학습 (Claude 의존도 감소)
- 구독 모델 도입 (월정액 상담권)
- 파트너 상담사 모집 (수익 분담 모델)
